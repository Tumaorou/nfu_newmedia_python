# 简介 
手机游戏王者荣耀的英雄资料查询


## 输入：
用户输入英雄名
## 输出：
用户得到输出结果为：英雄的技能和背景故事
## 从输入到输出，本组作品使用了：
### 模块
* [requests](http://docs.python-requests.org/zh_CN/latest/user/quickstart.html)
* [BeautifulSoup](http://cuiqingcai.com/1319.html)
### 数据
* [王者荣耀英雄数据](http://pvp.qq.com/web201605/herolist.shtml)
### API
* [王者荣耀游戏官网英雄资料](http://pvp.qq.com/web201605/herodetail/166.shtml)
